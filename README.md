Cucumber-JVM-Parallel
=====================

Example project showing running of Cucumber-JVM tests in parallel

To run, you will need to install maven, then:

mvn clean install

A report will be generated at /target/cucumber-report/index.html

FOR A MORE UP TO DATE VERSION OF THIS CODE SEE https://github.com/opencredo/test-automation-quickstart and associated blog posts https://opencredo.com/test-automation-quickstart-framework/ and https://opencredo.com/test-automation-concepts-parallel-test-execution/
